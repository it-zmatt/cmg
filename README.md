## Prerequisites
Ensure you have the following installed:

### Mac:
- PHP (Check with `php -v`, install with `brew install php` if needed)
- Built-in PHP server (included with macOS)

### Windows:
- **Option 1**: Laragon (Recommended for a lightweight environment)
- **Option 2**: XAMPP
